#### Localhost
나의 호스트 환경이다.
참고만 하면 되고 실제로는 가상머신에 Ubuntu 를 띄우는게 핵심.

 - Windows 10
 - CPU i7-9700k (8 core)
 - 64GB RAM

#### Oracle VirtualBox
 - 6.1.22 버전
 - 확장팩도 설치했음
 - 로컬에 WSL2 도커가 있어서 가상머신 런 안되는 문제가 있음 ("1. Docker" 에 해결책 설명해둠)

#### Ubuntu
 - Ubuntu Server 20.04LTS 이미지로 가상머신 설치
 - Master, Node1, Node2 이렇게 3개 설치
 - 설정 - 네트워크 - 어댑터에 브리지 (이래야 localhost 에서 접속 가능)
 - **프로세스 최소 2이상** 이어야된다 !
